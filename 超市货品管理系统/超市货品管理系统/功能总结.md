# 超市货品管理系统 - 功能总结

## 系统概述
本系统是一个基于Web的超市货品管理系统，支持商品信息管理、库存管理、销售统计等功能。前端使用HTML+JavaScript，后端使用Flask API，数据存储使用JSON文件和浏览器LocalStorage。

---

## 📊 统计面板功能

### 当前统计项
| 统计项 | 说明 |
|--------|------|
| 商品总数 | 系统中所有商品的数量 |
| 库存总量 | 所有商品的库存数量总和 |
| 已售数量 | 所有商品已售出的数量总和 |
| 库存价值 | 库存商品的总价值（库存×价格） |
| 库存预警 | 库存低于预警值的商品数量 |
| 在售商品 | 状态为"在售"的商品数量 |

## 🔍 搜索与筛选功能

### 搜索功能
- **搜索范围**：产品号、产品名称、品牌、供应商
- **搜索方式**：实时搜索，输入后自动过滤结果

### 筛选功能
| 筛选类型 | 选项 |
|----------|------|
| 分类筛选 | 饮料、方便食品、乳制品、零食、调味品、水果、蔬菜、日用品、生鲜、冷冻食品、粮油、酒水、其他 |
| 状态筛选 | 全部状态、在售、下架 |
| 排序方式 | 默认排序、价格从低到高、价格从高到低、库存从少到多、库存从多到少、销量从高到低、名称A-Z |

## 📦 商品信息字段

### 基本信息
| 字段 | 说明 | 必填 |
|------|------|------|
| 产品号 | 商品唯一标识，可自动生成或手动输入 | 否（自动生成） |
| 产品名称 | 商品名称 | 是 |
| 分类 | 商品分类 | 是 |
| 品牌 | 商品品牌 | 否 |
| 规格 | 商品规格（如500ml、1kg） | 否 |

### 价格与库存
| 字段 | 说明 | 必填 |
|------|------|------|
| 价格 | 商品单价（元） | 是 |
| 库存数量 | 当前库存量 | 是 |
| 已售数量 | 已销售的数量 | 否（默认0） |
| 新入库 | 新入库的数量 | 否（默认0） |
| 库存预警值 | 库存低于此值时显示预警 | 否（默认10） |

### 其他信息
| 字段 | 说明 | 必填 |
|------|------|------|
| 产地 | 商品产地 | 否 |
| 保质期 | 商品保质期 | 否 |
| 供应商 | 供应商名称 | 否 |
| 商品描述 | 商品详细描述 | 否 |
| 商品图片 | 商品展示图片（Base64格式） | 否 |
| 商品状态 | 上架/下架状态 | 否（默认上架） |

## 🖼️ 图片功能

### 图片上传
- **上传方式**：点击上传区域选择文件，或拖拽图片到上传区域
- **支持格式**：JPG、PNG、GIF
- **建议尺寸**：400×300像素
- **存储方式**：转换为Base64格式存储在数据中

### 图片显示
- **卡片视图**：图片显示在卡片顶部，高度200px
- **表格视图**：缩略图显示，尺寸50×50px
- **详情页**：大图显示，尺寸200×200px
- **无图片时**：显示对应分类的图标

## 🔧 商品操作功能

### 添加商品
- 打开添加商品弹窗
- 填写商品信息（支持图片上传）
- 自动生成产品号或手动输入
- 保存后自动刷新列表

### 编辑商品
- 点击编辑按钮打开编辑弹窗
- 显示当前商品所有信息
- 可修改任何字段（包括图片）
- 保存后更新商品信息

### 删除商品
- 点击删除按钮打开确认弹窗
- 显示商品名称确认
- 删除后不可恢复
- 自动刷新列表和统计

### 商品详情
- 点击商品卡片查看详情
- 显示完整商品信息
- 可直接进行出库、入库、编辑操作

---

## 📥 出库/入库功能

### 出库功能
- 选择出库商品
- 显示当前库存
- 输入出库数量（不能超过库存）
- 可添加备注信息
- 出库后库存减少，已售增加

### 入库功能
- 选择入库商品
- 显示当前库存
- 输入入库数量
- 可添加备注信息
- 入库后库存增加，新入库增加

---

## 🏷️ 状态与标签

### 商品状态
- **在售(active)**：正常销售的商品
- **下架(inactive)**：暂停销售的商品
- 通过开关按钮控制状态

### 自动标签
| 标签 | 显示条件 |
|------|----------|
| 新品 | 新入库数量 > 0 |
| 库存紧张 | 库存 < 库存预警值 |
| 已下架 | 商品状态为inactive |

## 💾 数据存储

### 前端存储
- 使用浏览器LocalStorage存储商品数据
- 键名：`supermarket_products`
- 支持离线使用

### 后端存储
- 使用JSON文件存储（products.json）
- 支持Flask API接口操作
- 数据格式与前端一致

## 🔌 API接口列表

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/products` | GET | 获取商品列表（支持筛选、排序） |
| `/api/products/<id>` | GET | 获取单个商品详情 |
| `/api/products` | POST | 添加新商品 |
| `/api/products/<id>` | PUT | 更新商品信息 |
| `/api/products/<id>` | DELETE | 删除商品 |
| `/api/products/<id>/sale` | POST | 商品出库 |
| `/api/products/<id>/restock` | POST | 商品入库 |
| `/api/categories` | GET | 获取所有分类 |
| `/api/stats` | GET | 获取统计数据 |

## 使用方式

### 纯前端使用
直接在浏览器中打开 `index.html` 文件即可使用，数据保存在浏览器LocalStorage中。

### 后端服务使用
运行 `python app.py` 启动Flask服务，访问 `http://localhost:5000` 使用完整功能。
