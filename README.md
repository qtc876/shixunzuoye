# 超市货品管理系统

简洁高效的超市货品管理工具，支持商品管理、库存管理、销售统计等功能。

## 在线访问

**公网地址**：https://chaoshi-guanqi.netlify.app

直接点击链接即可使用，无需安装任何软件。

## 功能特性

### 数据统计
- 商品总数统计
- 库存总量统计
- 已售数量统计
- 库存价值计算
- 库存预警提示
- 在售商品统计

### 商品管理
- 添加商品（支持上传图片）
- 编辑商品信息
- 删除商品（带确认）
- 查看商品详情
- 商品上下架控制

### 库存管理
- 商品出库（减少库存，增加已售）
- 商品入库（增加库存）
- 库存预警设置
- 自动标签（新品、库存紧张、已下架）

### 搜索与筛选
- 关键词搜索（产品号、名称、品牌、供应商）
- 分类筛选（13种分类）
- 状态筛选（全部/在售/下架）
- 多种排序方式（价格、库存、销量）

### 视图展示
- 卡片视图（直观展示）
- 表格视图（信息全面）
- 分页显示（每页12条）
- 空状态提示

## 商品信息字段

| 字段 | 说明 |
|------|------|
| 产品号 | 唯一编号（P001格式） |
| 商品图片 | 支持上传本地图片 |
| 产品名称 | 商品名称 |
| 品牌 | 品牌信息 |
| 分类 | 饮料、方便食品、乳制品、零食等13种 |
| 规格 | 如550ml、500g |
| 价格 | 单价（元） |
| 库存 | 当前库存 |
| 已售 | 已销售数量 |
| 新入库 | 新入库数量 |
| 预警值 | 库存预警阈值 |
| 产地 | 商品产地 |
| 保质期 | 保质期说明 |
| 供应商 | 供应商名称 |
| 描述 | 商品详细描述 |
| 状态 | 在售/下架 |

## 使用说明

### 本地运行
1. 下载 `index.html` 文件
2. 直接用浏览器打开即可使用

### 在线使用
访问：https://chaoshi-guanqi.netlify.app

### 数据存储
- 数据存储在浏览器本地（localStorage）
- 首次使用会自动加载10条示例数据
- 刷新页面数据不会丢失
- 清除浏览器缓存会重置数据

## 技术栈

- **前端**：原生 HTML + CSS + JavaScript
- **UI设计**：现代扁平化风格，自适应响应式布局
- **数据存储**：浏览器 localStorage
- **部署平台**：Netlify

## 部署方式

### Netlify 拖拽部署（推荐）
1. 访问 https://app.netlify.com/drop
2. 将 `index.html` 文件拖入页面
3. 自动生成公网链接

### GitHub Pages
1. 创建 GitHub 仓库
2. 上传 `index.html`
3. 在仓库设置中启用 GitHub Pages

### 自建服务器
1. 使用任何 Web 服务器托管 `index.html`
2. 确保服务器支持静态文件
